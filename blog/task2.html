Proxy Server

Proxy server là một server trung gian giữa client và server. Nó cung cấp các mức chức năng, bảo mật và quyền riêng tư tùy thuộc vào nhu cầu người dùng.
Khi dùng proxy server, request sẽ chuyển đến proxy server trước khi đến web server. Sau đó, web server gửi lại một response tương ứng đến proxy server, proxy server sẽ chuyển tiếp dữ liệu nhận được cho client browser.

Operation

Mỗi thiết bị kết nối mạng đều có một địa chỉ IP riêng. Địa chỉ IP được xem là địa chỉ để xác định danh tính của các thiết bị khi giao tiếp qua internet.

Proxy server cơ bản là một máy tính có địa chỉ IP riêng. Khi client gửi một request đến web server, request này sẽ đến proxy server trước tiên. Proxy server sẽ thay client gửi request, thu thập response và gửi lại chúng về client browser.

Trong quá trình chuyển tiếp dữ liệu, proxy server có thể thay đổi dữ liệu gửi đi nhưng vẫn giữ lại thông tin mà client cần. Proxy server có thể thay đổi địa chỉ IP, vì thế, web server sẽ không thể xác định được danh tính client. Proxy server có thể mã hóa dữ liệu của bạn khi chuyển tiếp. Proxy server cũng có thể chặn truy cập đến những trang web nhất định dựa vào địa chỉ IP.

Benefit

  • Kiếm soát việc dùng internet: Thiết lập một proxy server để chặn hoặc chuyển hướng một số trang web nhất định, giám sát lịch sử duyệt web.
  • Tiết kiệm băng thông và tăng tốc độ truy cập: Proxy server có thể lưu vào bộ nhớ cache những trang web phổ biến. Khi truy cập vào một trang web, proxy server sẽ kiểm tra lại xem có bản copy hay không và gửi cho client. Điều này có nghĩa là nếu hàng trăm user truy cập vào một trang web nào đó, proxy server sẽ chỉ gửi một request đến trang web đó.
  • Riêng tư: Vài proxy server sẽ thay đổi địa chỉ IP và một vài thông tin khác chứa trong web request.
  • Bảo mật: Proxy server mã hóa web request, chặn các trang web có mã độc.
  • Truy cập vào các nguồn tài nguyên bị hạn chế.
